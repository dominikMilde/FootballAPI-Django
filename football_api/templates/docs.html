Docs - football API

RESOURCE Country

GET /api/country/{id}/ --> returns info about country with given id, 200 when OK, 404 when country with id doesn't exist
ex.

GET /api/country/1/

returns 

                {
                    "id": 1,
                    "name": "Croatia",
                    "name_short": "CRO",
                    "population": 3892135,
                    "currency": "HRK"
                }
PUT /api/country/{id}/ --> updates info about country with given id, 200 when OK, 404 when country with id doesn't exist, 400 when wrong arguments passed
ex.

PUT /api/country/1/

recieves 

                {
                    "name": "Croatia",
                    "name_short": "CRO",
                    "population": 3791252,
                    "currency": "EUR"
                }
DELETE /api/country/{id}/ --> deletes info country with given id, 204 when OK, 404 when country with id doesn't exist
ex.

DELETE /api/country/1/

return 

                {
                    "error": "Country doesnt exist"
                }
POST /api/country/ --> creates country with info given in body, 200 when OK, 400 when invalid info given. Creates ID on backend.
ex.

POST /api/country/

recieves 

                {
                    "name":"Croatia",
                    "name_short":"CRO",
                    "population": 3892135,
                    "currency": "HRK"
                }

In case of insufficient info, returns hint:

                {
                    "population": [
                        "This field is required."
                    ]
                }
GET /api/country/list/ --> gets list with all stored countries, 200 when OK
ex.

GET /api/country/list/

returns 

                [
                    {
                        "id": 1,
                        "name": "Croatia",
                        "name_short": "CRO",
                        "population": 3892135,
                        "currency": "HRK"
                    },
                    {
                        "id": 2,
                        "name": "Germany",
                        "name_short": "DE",
                        "population": 83130652,
                        "currency": "EUR"
                    }
                ]
